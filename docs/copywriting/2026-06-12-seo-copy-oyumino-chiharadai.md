## 3. 反映時の注意

1. **メタ説明文の貼り込みは1ページずつ**。`<meta name="description">` と JSON-LD（LodgingBusiness / FAQPage）の両方を更新する
2. FAQを追加したら **faq.html の FAQPage 構造化データにも同じQ&Aを追記**（リッチリザルト対象）
3. ちはら台の【要確認】が埋まるまで、未確定の数字・設備をサイトに出さない
4. 反映後は Google Search Console で再クロールをリクエストする【プロパティ登録があるか要確認】
